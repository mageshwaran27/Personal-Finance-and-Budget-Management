💰 Personal Finance & Budget Management
A Java-based application connected to SQL Plus via JDBC that helps users track expenses, manage budgets, and plan savings. The system provides insights, reminders, and reports to support smarter financial decisions and long-term stability.

✨ Features
📊 Expense Tracking – Record and categorize daily expenses.

🛠️ Budget Management – Set monthly/annual budgets and monitor progress.

💡 Savings Planner – Suggests savings goals based on spending habits.

📂 Database Integration – Stores user data securely in SQL Plus.

📑 Reports & Insights – Generate summaries and charts for better financial planning.

⚙️ Tech Stack
Language: Java

Database: SQL Plus

Connectivity: JDBC

🚀 How It Works
Users input income and expenses.

Data is stored in SQL Plus via JDBC.

The app analyzes spending patterns.

Reports and alerts help users stay on track with budgets and savings.

📌 Use Cases
Manage personal or family budgets.

Track recurring expenses and bills.

Plan savings for future goals.

Gain insights into financial health.
